# 📦 Pack Cours Programmation Mobile

Ce pack regroupe tous les supports de cours, TD, TP et projets pour la formation en **programmation mobile Kotlin / Ionic / Flutter**.

## 📂 Contenu du pack

- **TD_complet.pdf**  
  Séances 0 → 8 avec TD Kotlin & Ionic en parallèle + annexe sur les commandes terminal.

- **TP_corriges_enrichis.pdf**  
  Corrigés complets des travaux pratiques (Kotlin, Android).

- **cours_kotlin_ONIFRA_INFOL3.pdf**  
  Cours sur les bases de Kotlin avec historique et exemples de code.

- **MiniProjets_Corrections_Full.pdf**  
  Mini-projets corrigés avec implémentations Kotlin / Java / Ionic / Flutter.

- **Compilation_Deploy.pdf**  
  Guide pratique pour compiler vos apps et les déployer sur smartphone Android (Kotlin, Ionic, Flutter).

## 🚀 Utilisation

1. Lire **cours_kotlin_ONIFRA_INFOL3.pdf** pour la base du langage.  
2. Suivre les **TD_complet.pdf** séance par séance.  
3. Faire les exercices et consulter les corrections dans **TP_corriges_enrichis.pdf**.  
4. Explorer les **MiniProjets_Corrections_Full.pdf** pour des exemples complets d’applications.  
5. Compiler et tester vos apps sur smartphone en suivant **Compilation_Deploy.pdf**.
